# PRODIGY_DS_04

## Task 04 - Sentiment Analysis on Social Media Data

This project analyzes and visualizes sentiment patterns in social media data to understand public opinion and attitudes towards different topics and brands.

### Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

### Dataset

Twitter Training Dataset

### Project Steps

1. Loaded the social media dataset.
2. Assigned column names.
3. Analyzed sentiment categories.
4. Visualized sentiment distribution using a bar chart.
5. Saved the output graph.

### Output

* Sentiment Distribution Bar Chart (`sentiment_distribution.png`)

### Conclusion

The project successfully analyzed sentiment patterns in social media data and visualized the distribution of sentiments, helping to understand public opinions across different topics and brands.
